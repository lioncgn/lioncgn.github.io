Python - 属性访问顺序

属性访问优先级的顺序如下：

  • 类属性
  • 数据描述符 (实现了__get__ 和__set__方法的描述符)
  • 实例属性
  • 非数据描述符(仅仅实现了__get__方法的描述符)
  • 默认为__getattr__

针对描述符，注意事项如下：

  • 描述符本身应定义成新式类，被代理的类也应该是新式类
  • 把描述符定义为被代理类的类属性

总之，类的实例访问属性的顺序如下：

  • 数据描述符
  • 实例自己的__dict__
  • 非数据描述符
  • 类或者基类的__dict__
  • 如果自己定义了 __getattr__ ,就调用它，否则，抛出AttributeError异常

自己千万不要自己实现__getattribute__方法.
