ubuntu 使用docker

1.安装

1. 第一种安装方法： 使用官方安装脚本自动安装

  1. sudo curl -fsSL https://get.docker.com | bash -s docker --mirror Aliyun github地址

  2. docker守护程序始终以root用户身份运行 ,为了每次不用敲 sudo,将用户添加到docker用户组中

    sudo usermod -aG docker $USER ,然后重启系统才能生效

  3. 检查安装版本命令： sudo docker version

2. 第二种安装方法：手动安装，参考网址

  1. 卸载可能存在的旧版本命令

`sudo apt-get remove docker docker-engine docker-ce docker.io `

  1. sudo apt update

  2. sudo apt-get install -y apt-transport-https ca-certificates curl software-properties-common

  3. 添加Docker官方GPG密钥

    curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

  4. 检查GPG Key信息是否正确： `sudo apt-key fingerprint 0EBFCD88

  5. 添加源信息

    sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"

  6. sudo apt update

  7. 确认Docker源是否添加成功
    sudo apt-cache madison docker-ce

  8. 安装最新版本的Docker CE : sudo apt-get install -y docker-ce

  9. 如果需要安装某个指定版本的Docker CE，而不是最新版本，可用下面命令列出可用的版本列表信息：sudo apt-cache madison docker-ce

  10. 从上面命令中寻找版本字符串，使用此命令 安装指定版本sudo apt-get install docker-ce=[版本字符串]

  11. 查看docker服务是否启动
    systemctl status docker

3. 镜像加速

  1. 使用阿里云提供的个人镜像加速服务： 阿里云

  2. 配置源加速器，便于从dochub上快速拉取镜像

  1. 设置之后，通过命令查看docker配置加速器是否生效：

    docker info， 是否有 Registry Mirrors配置信息；

4. 基于Dockerfile 文件构建自己的镜像过程

  1. 编写Dockerfile文件

  2. 执行构建命令
    docker build -t csphere/php-fpm:5.4 .
    参数解释：
    -t ,代表要构建的镜像名, “.“, 表明Dockerfile所在的路径

  3. 查看本地的镜像命令
    docker images

  4. 使用创建的镜像启动容器

    docker run -d -p 8080:80 --name website csphere/php-fpm:5.4
    参数解释：
    -d:后台运行

    -P:随机端口映射

    -p:将容器的80端口映射绑定到本机的8080端口上，四种格式（未完全验证）

    ip:hostPort:containerPort

    ip::containerPort

    hostPort:containerProt

    containerPort

    –name:指定容器的名称，没有指定这个参数，将默认被指定；
    后面的csphere/php-fpm:5.4 表示基于此镜像

  5. 查看当前运行的容器命令
    docker ps
    如果查看所有的可以通过 docker ps -a

  6. 以交互形式进入容器里执行命令

    docker exec -it website /bin/bash
    退出容器： exit

  7. 删除容器命令 docker rm container id
    使用这条命令需要容器不是处于 Up状态，如果是Up状态，需要使用docker rm -f

  8. Dockerfile 命令

    • FROM 必须是第一个命令

    • MAINTAINER 镜像维护者信息

    • COPY|ADD 添加本地文件到镜像中 (这两个命令有区别)

    • WORKDIR 工作目录

    • RUN 构建镜像命令

    • CMD 启动容器时调用

    • ENTRYPOINT 设置容器初始化命令，使其可执行

    • ENV 设置的环境变量

    • EXPOSE

8. docker概念

  • 仓库（Registry)
    • 公有仓库：Docker Hub, 阿里，网易等
    • 私有仓库：企业内部搭建
      • Docker Register, Docker官方提供的镜像仓库存储服务
      • Harbor, 是Docker Register的封装，提供友好的Web UI界面，角色和用户权限管理，用户操作的审计等
      • 镜像访问地址形式： registry.devops.com/demo/hello:latest, 若没有前面的url,默认寻找Docker Hub中的镜像，若没有tag标签，使用latest作为标签；
