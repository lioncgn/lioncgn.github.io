4. Column 的常用参数

  • primary_key
  • autoincrement # 对于主键自动为True
  • nullable 默认为True
  • default 设置默认值
  • onupdate=datetime.now :当修改带这个参数的字段时，这个字段的此属性将自动更新时间；
  • name 参数自定义设置字段名
